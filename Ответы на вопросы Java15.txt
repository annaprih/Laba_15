1.Что такое Maven? Для чего создан Maven?
Это программный продукт для сборки проектов, который может быть встроен в IDE и имеет интерфейс консоли.

2.Как узнать какую версию Maven вы используете?
В командной строке прописать mvn --version или посмотреть в IDE.

3.Какая структура каталогов в Maven?
Папка src  включает в себя папки main и test, main содержит папки java и resources, test для файлов тестов на testNG и UnitTest.

4.Что такое pom.xml?
Файл конфигурации сборки проекта.

5.Какую информацию содержит pom.xml?
Настройки сборки, лицензии, модули проекта.

6.Что такое супер POM?
Это конфигурационный файл, который содержит настройки сборкипо умолчанию, фал pom.xml находящийся в проекте наследует supperpom.xml.

7.Какие элементы необходимы для минимального POM?
groupId
artifactId
version

8.Что такое артефакт?Что является полным именем артефакта?
Выходной файл упаковки. Полное имя - groupId + artifactId + version.

9.Что такое зависимости в Maven?
Подключение библиотек к проекту.

10.Что такое плагин в Maven?
Библиотеки,которые добавляют фазы сборки проекта.

11.Что такое задача в Maven?
Добавление каккие-либо действий при выполнении фаз сборки. Тег <goal> может добавлять какие-либо задачи при определенных фазах сборки. 

12.Что такое репозиторий в Maven?Какие типы репозитория существуют в Maven?
Хранилище библиотек jar, war. Локальный, удаленный, центральный.

13.Какой порядок поиска зависимостей Maven?
Сразу на локальном, потом на центральном, потом в удаленном репозитории.

14.Назовите основные фазы жизненного цикла сборки Maven?
Compile, package, install, clean, deploy, test, site.

15.Что делает команда mvn site? Что делает команда mvn clean?
site - создает сайт с документацией о проекте.
clean - очищает проект.

16.Из каких фаз состоит жизненный цикл сборки Default (Build)?
validate, compile, test, package, install, deploy.

17.Какие типы плагинов существуют в Maven?
Плагины сборки и плагины отчетов.

18.Когда Maven использует внешние зависимости?
Если необходимые файлы не найдены ни в локальном, ни в центральном, ни в удаленном репозитории, тогда будут искаться внешние зависимости на других репозиториях.

19.Какая команда создает новый проект на основе архетипа?
mvn archtype:generate

20.Что такое транзитивная зависимость в Maven?
Необходимые библиотеки подключаются автомотически.
